# CSC316-data-viz-project
Group Course Project for Data Visualization on the web, using D3.

## Our Code

- All of our javascript code can be found under `./js` folder
- All of our css code can be found under `./css` folder
- All of our html is in the `index.html` which can be loaded to take you to our webpage (or you can visit our link)

## Published Website

- Deployed webpage found [here](https://dadler83.github.io/CSC316-data-viz-project/)

## 3rd Party Libs

- Not included directly with the source code, 3rd party libraries are linked in the `index.html` and in `./js/donut3d.js`
- 3rd Party libraries used:
  - [D3.js](https://d3js.org/)
  - [D3-3d.js](https://github.com/Niekes/d3-3d)
  - [Bootstrap](https://getbootstrap.com/)

## Data
